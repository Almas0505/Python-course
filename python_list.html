📋 Python Тізімдер (Lists)

🗂️

Бірнеше элементтерді сақтауға арналған өзгермелі деректер құрылымы

Тізім дегеніміз не?

Тізім - бұл реттелген элементтер жинағы. Python-да тізімдер квадрат жақшамен жазылады және әртүрлі деректер түрлерін сақтай алады.

# Бос тізім empty_list = [] # Сандармен тізім numbers = [1, 2, 3, 4, 5] # Строкалармен тізім cities = ["Алматы", "Астана", "Шымкент"] # Аралас тізім mixed = ["Python", 3.9, True, [1, 2]]

  • Тізімдер өзгермелі (mutable) - элементтерді өзгертуге болады
  • Әртүрлі деректер түрлерін сақтай алады
  • Индекстер 0-ден басталады
  • Қайталанатын элементтер болуы мүмкін

🔧 Негізгі Операциялар

Элементтерге қол жеткізу

fruits = ["алма", "банан", "алмұрт"] # Индекс арқылы print(fruits[0]) # алма print(fruits[-1]) # алмұрт (соңғы элемент) # Кесінділер (slicing) print(fruits[0:2]) # ['алма', 'банан'] print(fruits[1:]) # ['банан', 'алмұрт']

Элементтерді өзгерту

numbers = [1, 2, 3] # Жалғыз элементті өзгерту numbers[1] = 10 print(numbers) # [1, 10, 3] # Бірнеше элементті өзгерту numbers[0:2] = [5, 6] print(numbers) # [5, 6, 3]

⚡ Тізімнің Пайдалы Әдістері

append()

Соңына элемент қосады

fruits = ["алма"] fruits.append("банан") # ["алма", "банан"]

insert()

Белгілі позицияға қосады

fruits.insert(1, "алмұрт") # ["алма", "алмұрт", "банан"]

remove()

Элементті алып тастайды

fruits.remove("банан") # ["алма", "алмұрт"]

pop()

Элементті алып, қайтарады

last = fruits.pop() # last = "алмұрт", fruits = ["алма"]

sort()

Тізімді сұрыптайды

numbers = [3, 1, 4, 2] numbers.sort() # [1, 2, 3, 4]

reverse()

Тізімді кері бұрады

numbers.reverse() # [4, 3, 2, 1]

🎉 Қорытынды

Python тізімдері - бұл өте қуатты және икемді деректер құрылымы. Олардың көмегімен сіз:

  • ✅ Көптеген деректерді ұйымдасыра аласыз
  • ✅ Деректерді тиімді өңдей аласыз
  • ✅ Күрделі алгоритмдер жаза аласыз
  • ✅ Код жазуды жеңілдете аласыз

class GradeManager: def __init__(self): self.students = [] self.grades = [] def add_student(self, name, grade): self.students.append(name) self.grades.append(grade) def get_average(self): return sum(self.grades) / len(self.grades)
